Открыть pic.fig для того чтобы видеть графики

Данный проект позволяет проводить считывание данных и Калмановскую фильтрацию к результатам измерений.

matlab.mat - Файл сырых измерений
X,Y,Z.mat - отсортированные файлы по X,Y,Z из matlab.mat для кода python

1. Запускаю read_kalman.py  - проводится фильтрация результатов измерений X,Y,Z, получаю файл Ant1_two_antennas_XYZ_real_time.mat
2. Результаты дисперсии от проведенного измерения хранятся в Результаты дисперсии.PNG 
3. Visualization.m - считывает результаты измерений из matlab.mat и отфильтрованные данные из Ant1_two_antennas_XYZ_real_time.mat, отображает на рисунке (здесь сохранены под pic.fig)
